# 单色系比例规则 92-6-2

## 规则

一条高级感 showreel 的色板，**时间占比**应该符合：

- **92% 主色域**：2 个色（1 个 accent 主色 + 1 个 bg 底色）贯穿全片
- **6% 点缀色**：1 个第三色，出现 1-2 次、每次 0.5-1s，用于"视觉惊喜点"
- **2% 反转色**：1 个强对比色（通常是黑或白），仅在"视觉高潮/drop 瞬间"出现 1 次、< 1s

## 从哪来

Aftermagics × Datalyr 17.5s 里：
- 15s 是暖橙 `#EC6C1C` + 米白 `#F1EAE2`（92%）
- 6s 出现紫 `#7965CA`（1s，占 6%）
- 11s 出现黑 `#030303`（1s，占 6%——这里其实是 6% 不是 2%）

**注：单样本比例不精确**，"92-6-2" 是名字取整，等 3+ 样本再校准数字。

## 反例

- ❌ 主色 5 种以上 → 拼贴感
- ❌ 反转色出现多于 2 次 → 失去"惊喜"，变成噪声
- ❌ 点缀色和主色对比度太低 → 用户感知不到

## Agent 使用指令（未来喂 prompt 用）

> 生成分镜时，先确定 3 种颜色：主色 A、底色 B、点缀色 C。
> - 每一镜的主导色只能是 A 或 B。
> - C 只允许出现在 1-2 镜，每镜 durationSec ≤ 1.0s。
> - 如果方向卡里给了反转色 D，D 只能出现在"drop 镜"上，且全片仅 1 次。

## 待验证问题

- [ ] 92-6-2 具体数字是否稳定？（需要 3+ 样本）
- [ ] "反转色"是否所有 showreel 都有？还是只有"品牌 outro"型的才有？
- [ ] 点缀色出现的位置是否有规律（比如都在片长 30-40% 位置）？
