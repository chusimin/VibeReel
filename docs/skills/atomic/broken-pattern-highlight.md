# 打破视觉规律的时刻 = 观众记忆点

## 观察

Aftermagics × Datalyr 里：
- 前 10 秒**所有元素都是"圆角矩形卡片"**（流程节点、标题框、通知框）
- 第 11 秒突然出现**橙色圆形 logo**（形状规律被打破）
- 这一秒也是全片唯一的"drop 瞬间"（黑幕 + scale）

**结论**：**视觉规律建立 → 精确一次的打破 = 高潮**。观众记住的往往就是这一秒。

## 规则

一条 showreel 必须有**一个可识别的视觉母题**（形状/结构/色块/图案），并且**在 60-75% 时长位置精确打破一次**：

**建立母题的方法**：
- 反复出现的形状（圆角矩形、菱形、圆形...）
- 反复出现的结构（居中构图、九宫格、双列...）
- 反复出现的颜色比例
- 反复出现的字体家族

**打破的方法**（选一个，只用一次）：
- 形状换：矩形 → 圆形
- 构图换：居中 → 满屏 / 特写
- 色反转：亮 → 暗 / 明 → 黑
- 尺度换：小 → 巨大

## 反例

- ❌ 没有母题，每镜都不一样 → 无"规律"可打破，"打破"无效
- ❌ 母题被打破多次 → 变成新的规律，失去高潮效果
- ❌ 打破发生在开头或结尾 → 位置错，没有"drop"感

## Agent 使用指令

> 生成分镜时：
> - 显式声明这条视频的 `visualMotif`（母题类型 + 描述）
> - 至少 60% 的镜必须体现这个母题
> - 选择一个镜作为 `dropShot`，位置在时长 60-75% 处
> - `dropShot` 必须明确打破 `visualMotif`（在字段里说明怎么打破）

## 待验证

- [ ] 是否所有 showreel 都需要一个 drop 时刻？还是"生长动画型"独有？
- [ ] 打破位置 60-75% 是否稳定？
