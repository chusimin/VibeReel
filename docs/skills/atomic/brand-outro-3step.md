# 品牌收束三段式 (Brand Outro 3-Step)

## 观察

Aftermagics × Datalyr 最后 4.7s（占全片 27%）的结构：

| 时间 | 内容 | 时长 |
|---|---|---|
| 12.8-14.4s | Logo + "Datalyr" 品牌名（fade-in） | 1.6s |
| 14.4-16.0s | Slogan "Faster insights, Better decisions"（typewriter 逐词） | 1.6s |
| 16.0-17.5s | **CTA "Get started at Datalyr.com" + 静止** | 1.5s |

**三段渐次由动到静**，最后 1.5s 完全静止让用户"记住网址"。

## 规则

品牌 / 产品 showreel 的收束段必须包含**三个信息层，节奏由快转慢至静止**：

**第 1 层 · 身份识别**（1-2s）
- Logo + 品牌名 fade-in
- 只有品牌视觉基因，没有额外信息

**第 2 层 · 价值主张**（1-2s）
- Slogan / 价值卡片，字数 ≤ 12 词
- 用 typewriter / 逐词淡入的方式呈现，让用户"逐字读"

**第 3 层 · 行动召唤**（≥ 1.5s，且必须静止）
- CTA：URL / 应用商店 / "Try now" 类
- **必须在最后 1s 完全静止**（无任何动画）
- 用户需要时间记住网址

**总时长约 4-6s**（约占 15-25s 视频的 25-30%）。

## 反例

- ❌ Logo 出来后立即换 slogan → 用户没时间识别品牌
- ❌ CTA 还在动 / 变色 → 用户来不及记住
- ❌ 三段挤在 2 秒内 → 全部信息糊在一起
- ❌ 用 kinetic text 花式动画呈现 URL → 记不住
- ❌ 只有 logo 没有 CTA → 用户不知道下一步做什么

## Agent 使用指令

> 生成分镜时，最后 3 个分镜必须遵循 outro-3step：
> - N-2 镜：只有 logo + 品牌名，durationSec 1.5-2s，动效仅限 fade
> - N-1 镜：slogan/价值主张，durationSec 1.5-2s，动效仅限 typewriter
> - N 镜（末镜）：CTA (URL / 按钮)，durationSec ≥ 1.5s，其中末 1s 必须静止
>
> 末镜 durationSec = 音乐结尾静默时长，两者对齐。
