# 视觉密度曲线：从来不密

## 观察

Aftermagics × Datalyr 的密度曲线：

```
极简 → 累加至"中" → 极简 → "中" → 极简收束
```

**核心洞察**：**从来没有"密"**。所有镜要么是"极简"（1-2 元素），要么是"中"（3-5 元素）。**没有 6+ 元素同时出现的镜**。

## 规则

一条 showreel 的密度**必须有起伏**，且**最高不能超过"中"**：

| 密度级别 | 元素数量 | 允许时长占比 |
|---|---|---|
| 极简 | 1-2 | 40-60% |
| 中 | 3-5 | 40-60% |
| **密** | **6+** | **禁止 (0%)** |

**曲线**应该是"极简 ↔ 中"交替，**在"起承转合"的段落切换点必须回到极简**。

## 为什么

- **高级感 = 呼吸感** = 有留白时刻
- 观众的注意力容量在短视频里非常有限，超过 5 个元素就"看不过来"
- 密度陡降 = 段落切换信号（比转场更强的分段感）

## 反例

- ❌ 屏幕塞满 8 个卖点 → 拼贴感、廉价感
- ❌ 密度一直"中" → 无呼吸，观众疲劳
- ❌ 密度从"密"陡降到"极简" → 混乱，观众失焦

## Agent 使用指令

> 生成分镜时：
> - 每一镜声明 `density: "minimal" | "medium"`（禁止 "dense"）
> - 相邻两镜密度必须不同（避免连续 "medium"）
> - 每 4 镜里至少有 2 镜是 "minimal"
> - "drop 镜" 和 "outro 镜" 必须是 "minimal"
