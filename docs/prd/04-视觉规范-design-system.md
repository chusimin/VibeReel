# VibeReel · 视觉规范 / Design System
> 子文档 · 隶属 [主文档](00-主文档-PRD.md)｜讲：网页 UI 的基调、颜色、字体、字号、间距、圆角阴影、关键组件、生成态｜上位结构见主文档「架构总览」。
> 维护提示：改本文不影响其他子文档；跨文档引用集中在文末「关联」。注意：本文只管网页 UI；视频成片本身的 look 由四件套的风格包决定，见 [07-智能体-agent](07-智能体-agent.md)。

> 适用范围：本文规范的是**网页 UI**（网站本身的基调、配色、组件）；**视频成片本身的 look** 由四件套的风格包决定，不在此（见 [07-智能体-agent](07-智能体-agent.md)）。

## 基调

- 亮色高 key、大留白、大字号。
- **单一强调色只做"火花"**。

## 字体

- 正文：`"Inter", system-ui, "PingFang SC", sans-serif`（用 `next/font/google` 加载 Inter）。
- 等宽：`ui-monospace`。

## 字号阶梯

| 用途 | 字号 / 字重 |
|---|---|
| 主标题 | `32/600` |
| 区块标题 | `20/600` |
| 正文 | `15/400` |
| 辅助 | `13/400` |
| 按钮 | `15/500` |

## 间距

- 基准：`4`。
- 常用：`8/12/16/24/32`。
- 内容区最大宽：`960`（媒体预览区可到 `1120`），居中。

| 元素 | 圆角 |

| 卡片 | `16` |
| 按钮 / 输入 | `10` |

**阴影**

| 层级 | 值 |
|---|---|

## 关键组件

### 类型选择卡

- 三张大卡（showreel / 教学 / 科普），每卡含一句定位 + 一张示意缩略。
- 选中描边强调色。
- 说明：各类型对应的「风格卡候选」属四件套配置，不在本文罗列，见 [07-智能体-agent](07-智能体-agent.md)。

### 画幅选择

- 三个 pill（16:9 / 9:16 / 1:1），默认 16:9 高亮。
- 选中实时改下方预览框比例。

### 素材上传区

- 拖拽框 + 已传缩略网格。
- 每项可删。
- 标注「可选，但传得越全成片越准」。

### 草稿卡（分镜闸门）

- 每镜一卡 = gpt-image 草稿图 + 文字 beat（序号 / 时长 / 画面描述 / 字幕）+ 角标「草稿·方向参考」。

### 预览

- 视频按所选画幅居中，最大高 `70vh`。

## 生成中状态（B3 专门设计）

- 分段闸门每个分镜位先占一张**骨架占位卡**（shimmer 微动 + 当前后端名 + 阶段文案，如「氛围镜·生成式渲染中」）。
- 渲完原位替换成 mp4 播放器，**不跳动、不整页刷新**。
- 失败则占位卡变红 + 「重试该镜」。
